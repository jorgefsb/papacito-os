# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in Papacito OS, please open a private security advisory on GitHub.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will respond within 48 hours and work to resolve the issue promptly.

## Security Considerations

Papacito OS is designed to be 100% local:

- ✅ No data leaves your machine
- ✅ No cloud services
- ✅ No API keys needed
- ✅ Your data is stored locally in SQLite

However, please note:
- Backup your data regularly
- Keep your system updated
- Use strong passwords if sharing your machine
